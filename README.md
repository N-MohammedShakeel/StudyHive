# StudyHive

A full-stack learning-platform project by N-Mohammed Shakeel.

## Repository Structure

| Folder               | Description                                                         | Link                                                                                          |
| -------------------- | ------------------------------------------------------------------- | --------------------------------------------------------------------------------------------- |
| **backend**          | The server-side part of StudyHive: API, database, business logic.   | [backend](https://github.com/N-MohammedShakeel/StudyHive/tree/main/backend)                   |
| **frontend**         | The client-side (UI) of StudyHive: user interface, web app.         | [frontend](https://github.com/N-MohammedShakeel/StudyHive/tree/main/frontend)                 |
| **frontend/src/api** | API call layer in the frontend: wrappers, endpoints, request logic. | [frontend/src/api](https://github.com/N-MohammedShakeel/StudyHive/tree/main/frontend/src/api) |
